Jessy
=======

Get value by object property.

## Install

`npm i jessy --save`

## Hot to use?

```js
var jessy   = require('jessy');

jessy('hello.world', {
    hello: {
        world: {
            'could be used in browser as well'
        }
    }
});

// returns
'could be used in browser as well'

```
